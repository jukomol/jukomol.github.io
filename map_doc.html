For offline navigation of MT we will use raspberry pi 3B and GMouse gps. First of all, our task is to start GPSD. To do that simply install gpsd. open the terminal and enter

sudo apt-get install gpsd gpsd-clients python-gps
sudo systemctl stop gpsd.socket
sudo systemctl disable gpsd.socket

Now test, gpsd. Goto terminal and type

CGPS -s

Our next job is to download navit from here. Now install it manually, through terminal.
